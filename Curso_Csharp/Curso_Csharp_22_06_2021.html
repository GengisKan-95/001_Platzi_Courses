Hola mundo de un BackEnd en C sharp

¿Que se puede crear con C sharp?

Web Stack: HTML 5, CSS 3, JS, frameworks punto net Stack: C sharp, ASP, net Frameworks, Libreries.

¿Que es ASP.NET Core? Es un frameWork de codigo abierto que sirve para construir Apps.
Se basa en conjunto de paquetes nuguet.
Es multiplataforma, tiene buen rendimiento, es de codigo abierto, es escalable y modular.
Para mas informacion acerca de ASP visitar: https://docs.microsoft.com/es-es/aspnet/core/introduction-to-aspnet-core?view=aspnetcore-5.0

Pasos para realizar el BackEnd


1. Crear un nuevo proyecto Web con ASP Web Application, C sharp

2. Seleccionar el tipo de aplicacion web: En este caso se crea una plantilla Vacía con Web API

3. Lo primero que se debe crear es un Modelo, una estructura de datos. Esto se realiza creando una nueva clase

4. Se debe colocar el modelo público

5. Para el ejemplo se crean dos abributos con sus respectivos Setters y Getters. Los atributos son: id y name, realizar con propfull

6. Compilar el proyecto

7. Se debe crear un Controlador, para ello en la carpeta Controllers se crea un nuevo "New Scaffolded item"

7.1 Se selecciona la opción de Web API 2 Controller with Entity Framework

7.2 Se selecciona el modelo, se crea un data Context

7.3 Se selecciona un nombre para el controlador y se crea

7.4 Compilar y luego lanzar la aplicacion

8. Probar el servicio con cualquier cliente que sea capaz de conseguir APIs REST, app PostMan o extension JaSON

9. Al abrir el criente se coloca la URL que nos da el BackEnd

10. Entity framework luego de crear el modelo el crea la base de datos luego del primer Request

11. Aprender a usar el maquetado REST




Muchos conceptos desconocidos :( Investigar el tema de Json y XML. Se creó un controlador para administrar la base de datos.

Apps moviles de Windows Phone y Android


1. Ejecutar VSC como administrador

2. Clic derecho sobre la solucion, Agregar, Adicionar nuevo proyecto

3. Seleccionar "Windows -> Aplicacion en blanco -> Colocar nombre

4. Clic derecho al proyecto -> Establecer como proyecto predeterminado

5. Doble clic al archivo de "MainPage.xaml", la cual tiene un archivo XML y se puede editar visualmente

6. Crear un botón con ToolBox, cuadrar la resolución para celular

7. Abrir el panel de propiedades, cambiar el nombre

8. Luego de agregar la interfaz gráfica, clic derecho al proyecto y compilar, luego clic derecho e implementar

9. Lanzar la aplicacion para verifiar

10. Emular un celular y lanzar la aplicacion

11. Establecer un evento en las propiedades del botón con F4, activar el evento Tapped
